# Data files
* **destinations.tsv**: lists all the destinations supported by the LED destination signs. (Chinese & English)
* **lines-info.tsv**: lists whether the line name represents passenger service, the station names (in Chinese), and the service types (in Chinese).
* **lines-selection-lists.tsv**: lists all the station names available in each selection list of each line (in Chinese).
* **service-types.tsv**: Chinese-English translations of service types.
